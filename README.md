# E-Commerce Product Display

This is a basic e-commerce website that displays a list of products and allows users to add them to the cart.  
It is built using **HTML**, **CSS**, and **JavaScript**.

---

# Features

- Displays products dynamically using JavaScript  
- Each product includes an image, name, and price  
- "Add to Cart" button for each product (currently shows an alert message)
